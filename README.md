# Quick start guide
1. Create the virtual environment:
    ```
    $ python3 -m venv env
    ```
1. Activate the virtual environment:
    ```
    $ source env/bin/activate
    ```
1. Install the dependencies:
    ```
    (env) $ pip install -r requirements.txt
    ```
1. Run the server:
    ```
    (env) $ jupyter notebook notebooks/
    ```
